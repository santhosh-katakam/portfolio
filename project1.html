Petrol Station Website

Overview

This is a fully functional Petrol Station Website designed to modernize fuel station operations and provide users with a seamless experience to track and manage petrol services.

Features

  • Live fuel price tracker
  • Online fuel booking system
  • Admin dashboard to manage fuel and users

Technologies Used

  • React.js – Component-based dynamic user interface
  • Node.js – JavaScript runtime environment for server-side logic
  • MySQL – Relational database to store users, bookings, prices
